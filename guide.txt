in the terminal :

cd ~/quicklisp/local-projects/lisp-webapp
sbcl

in the sbcl :

(ql:quickload "lisp-webapp" :force t) 

or :

(ql:quickload :lisp-webapp)

then :

(lisp-webapp/frontend:build-assets)

then:

(lisp-webapp:start-app)

to stop server :

Ctrl + D or (lisp-webapp:stop-app)

to push :

git push -u origin main
